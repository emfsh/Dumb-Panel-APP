# v1.0.8

## 更新内容

### 修复

- 修复任务运行后首次进入实时日志页时，右上角错误显示“已完成”的问题
- 修复任务运行后首次进入实时日志页时，页面可能暂时看不到实时日志、返回再进入才正常的问题
- 修复实时日志页在首次连接阶段误用状态字段与日志流目标，导致运行状态与日志内容不同步的问题
- 修复任务实时日志页在进入后会把历史日志重复显示一遍的问题
- 修复面板日志页面读取错误字段导致始终显示“暂无日志内容”的问题

### 优化

- 优化任务实时日志页的初始化逻辑，首次进入会更稳定地等待日志就绪并接入正确日志流
- 优化实时日志页状态切换时机，运行中、完成、失败、等待日志等状态展示更准确
- 优化实时日志页对启动窗口期和历史重放的处理，减少假完成与重复追加

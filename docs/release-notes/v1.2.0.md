## 更新内容

### 新功能

- **任务分组管理**：分组 header 新增菜单，支持重命名、删除分组和批量添加任务到分组
- **任务分组排序**：长按分组 header 进入排序模式，拖拽调整分组显示顺序
- **环境变量排序**：长按变量列表进入排序模式，拖拽调整顺序并通过 API 同步到面板
- **日志批量管理**：长按日志项进入多选模式，支持批量删除和按天数清理旧日志
- **依赖状态筛选**：依赖管理页新增「已安装/失败」快速筛选，直观查看安装状态
- **面板版本显示**：主页服务器信息卡片显示面板版本号

### 优化

- **订阅表单完善**：新增覆盖本地修改、拉取后钩子、依赖说明、指定子目录字段，与面板完全对齐
- **白名单/黑名单分隔符统一**：从 `|` 改为逗号，与面板保持一致
- **未分组默认折叠**：任务未分组区域首次打开时默认折叠，减少列表干扰
- **APK 更新加速**：下载链接自动通过 GitHub 镜像加速，无需梯子即可更新
- **APK 复用**：已下载的安装包自动复用，不再重复下载

### 修复

- **暗黑模式编辑器**：修复暗黑模式下脚本编辑器文字不可见的问题（深色背景 + 深色文字）
- **依赖排队状态**：修复 `queued` 状态未处理导致显示为「已安装」的问题

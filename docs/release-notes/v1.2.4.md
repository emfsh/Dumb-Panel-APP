# v1.2.4

## 更新内容

### 新增

- **定时任务批量操作**：定时任务列表新增「批量」模式，支持批量运行、批量启用、批量禁用和批量删除，减少大量任务维护时的重复点击。
- **定时任务拖拽排序**：定时任务列表新增「排序」模式，可长按拖拽调整当前任务列表顺序，并在点击「完成」后保存排序。
- **环境变量值全屏编辑**：环境变量新建和编辑弹窗中，变量值输入框新增全屏编辑按钮，账号较多、Token 较长或多行内容较多时更方便编辑。

### 优化

- **主页任务概览直达任务页**：主页「任务概览」统计卡片支持点击直达定时任务页面，查看任务详情更顺手。
- **脚本管理操作菜单文案**：脚本管理相关操作菜单保持中文展示，删除、移动、复制、版本历史等入口更容易识别。

### 修复

- **脚本管理二级菜单显示不完整**：修复脚本/文件夹操作菜单在小屏设备上内容过多时无法看到底部「删除」按钮的问题，现在菜单会限制高度并支持上下滚动。
- **脚本管理无法手动删除单个脚本**：修复单个脚本的「删除」入口因菜单高度不足被遮挡的问题，现在可正常在脚本操作菜单中找到并执行删除。
- **日志清理全部无效**：修复运行日志右上角扫把选择「清理全部日志」时实际仍按默认保留天数清理的问题，现在会按当前筛选条件读取全部日志并批量删除。

### 说明

- 本版本主要面向 APP 端常用管理页面的交互补齐和体验修复，不新增数据库结构。
- 定时任务排序通过面板现有任务更新接口写入 `sort_order` 字段；批量任务操作使用面板后端要求的 `task_ids` 参数。

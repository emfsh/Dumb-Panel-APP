# v1.0.3

## 更新内容

### 功能对齐面板 v2.0.0

- 任务模型补充 `cron_expressions` 数组字段，任务卡片支持显示多条定时规则。
- 任务编辑表单 Cron 输入改为多行，每行一条规则；预设规则改为追加模式，不再覆盖已有规则。
- 订阅模型补充 `depend_on`、`hook_script` 字段，创建和编辑时可同步提交。
- 新增订阅停止拉取 API，订阅卡片在拉取中状态下显示红色停止按钮，可直接中止。
- 主页新增面板名称显示，有自定义标题时展示标题 + 欢迎语两行排版。

### Bug 修复

- 修复任务状态筛选器点击"运行中"后无法切回"全部"的问题（copyWith sentinel 模式）。
- 修复脚本查找跳转后无法看到匹配位置的问题，匹配时 2 秒琥珀色高亮选中并提示行号。
- 修复环境变量编辑/新建弹窗中 TextEditingController 未 dispose 导致的内存泄漏。
- 修复订阅 SSE 拉取日志页面可能重复创建连接的问题，重连前先关闭已有连接。
- 修复订阅列表加载失败时错误被吞没、用户无感知的问题，新增 error 状态字段。
- 修复登录页 TOTP 验证码输入只检查非空不检查 6 位长度的问题。

### 体验优化

- 应用锁触发时机优化：只在重新打开 App 时验证，切后台/前台不再频繁触发锁定。
- 任务卡片底部操作按钮（运行/暂停/编辑/更多）改为 36px 圆形带背景色，增大点击区域减少误触。

## 更新内容

### 新功能

- User 模型新增头像字段，设置页和首页均支持显示用户头像（无头像时保持首字母 fallback）。
- 设置页用户卡片下方显示当前连接的服务器地址。
- 日志页面和任务运行日志页面新增「复制全部」按钮，一键复制完整日志内容。

### 修复

- 修复日志内容无法长按选中复制的问题，将 `SelectionArea` + `RichText` 替换为 `SelectableText.rich`，支持原生文本选择和上下文菜单。
- 修复自动登录失效的问题：APP 冷启动时 token 过期后停留在服务器选择页而不是自动跳转登录页执行自动登录。
- 去掉 HTTP 仅允许局域网 IP / localhost 的限制，改为弹窗确认提示后允许连接，方便用户使用 HTTP 反代域名。

# v1.0.4

## 更新内容

### Bug 修复

- 修复全部列表页（共 11 处）RefreshIndicator 白屏问题：加载态和空态使用非可滚动组件导致下拉刷新失效和页面白屏。
- 修复日志页滑动筛选后白屏的问题（ScrollController 溢出 + RefreshIndicator 子组件不可滚动）。
- 修复安全设置页会话 Tab 和 IP 白名单 Tab 使用 Column 作为 RefreshIndicator 子组件导致无法下拉刷新。

### 体验优化

- 系统设置页移除未使用字段，消除编译警告。
- 修复 Switch 组件使用已弃用 `activeColor` 属性的警告。
- APK 构建改用固定 release 签名，支持覆盖安装升级，无需卸载旧版本。

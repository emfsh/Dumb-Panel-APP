# v1.2.3

## 更新内容

### 修复

- **设置页系统管理入口重复显示**：移除系统管理区域上方重复的「脚本 / 订阅 / 依赖 / 系统」快捷按钮，保留下方完整的管理列表入口，页面结构更清晰。
- **NAS 反代登录错误提示优化**：登录接口返回 403 时，不再笼统显示“网络错误”，会提示群晖、飞牛等 NAS 使用 Nginx Proxy Manager 或公网域名反代时需要升级面板到 v2.3.0 及以上；旧版可临时在 `config.yaml` 的 `cors.origins` 中加入完整公网地址。

### 说明

- 本版本主要面向 APP 端体验修复，不新增数据库结构或本地缓存结构。
- 面板端 NAS / Nginx Proxy Manager 反代 CORS 边界已在呆呆面板 v2.3.0 修复；旧面板版本如遇公网域名带端口登录 403，请先升级面板或临时补充 `cors.origins`。

# v1.2.5

## 更新内容

### 新增

- **环境变量右滑操作**：环境变量卡片支持右滑展开「启用 / 删除 / 禁用」三个按钮，列表默认界面更干净，常用操作也更顺手。
- **任务卡片右滑操作**：任务列表把启用、置顶、复制、编辑、删除等次要操作收纳到右滑区域，卡片底部只保留运行和停止，减少界面拥挤。

### 优化

- **任务 Cron 显示样式**：任务卡片里的定时表达式改为轻量信息卡，显示更清楚，不再像一串生硬文本。
- **任务订阅信息样式**：订阅来源改成带「订阅」标识的信息区，和任务主体内容区分更明显。
- **公共 Cron 列表组件**：统一优化定时任务公共展示组件，减少重复边框和杂乱标签，让 APP 端任务详情更耐看。
- **环境变量值放大编辑体验**：新建和编辑环境变量时，放大按钮不再跳转新页面，而是在当前弹窗里切换为大输入区，编辑完成后直接回到原表单。

### 修复

- **环境变量放大编辑不回填**：修复环境变量值在全屏输入界面保存后，返回新建/编辑表单时没有带回输入内容的问题。
- **环境变量编辑弹窗未收起**：修复旧版放大输入会在原弹窗上再打开新页面，导致新建/编辑弹窗仍停留在后面的体验问题。
- **大输入区误返回**：在环境变量大输入区按系统返回键时，现在会先回到原来的新建/编辑表单，不会直接关闭整个弹窗。

### 说明

- 本版本主要优化 APP 端环境变量和任务列表的移动端交互，不涉及后端接口和数据库结构变更。
- 环境变量删除仍保留确认弹窗，避免右滑操作误删重要变量。
- 环境变量的大输入区和原表单共用同一个输入控制器，因此多行 Token、账号列表或长文本内容不会再丢失。

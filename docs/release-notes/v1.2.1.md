# v1.2.1

## 更新内容

### 兼容呆呆面板 v2.2.17

- 登录页支持极验 V4 滑块验证码，开启验证码后可在 APP 内完成验证并继续登录。
- 依赖管理支持 Python 3.10 / 3.11 / 3.12 多版本环境查看、切换和默认版本设置。
- Python 依赖列表显示所属 Python 版本，并补充二进制部署与多版本依赖说明。
- 新建和编辑任务支持选择 Python 版本，默认超时调整为 `0`，表示永不超时。
- 通知渠道的邮件配置新增 SMTP SSL 开关。
- 数据备份下载接口适配新版 query 参数，修复备份包下载失败。
- 系统设置移除全局默认超时，新增代理地址说明、系统更新镜像源、二进制更新加速源配置。
- 订阅拉取后钩子和任务前后置钩子补充使用说明。

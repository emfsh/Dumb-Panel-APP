# v1.0.6

## 更新内容

### 修复

- 修复域名登录报"网络错误"：DioClient 新增协议自动切换拦截器，HTTP 失败自动尝试 HTTPS，反之亦然
- 服务器配置页智能协议选择：域名默认 HTTPS，IP/localhost 默认 HTTP
- 日志页面长按选择复制时滑动白屏：改用 SingleChildScrollView + SelectableText 替代 ListView.builder
- 任务实时日志不再强制滚动到底部，默认暂停，右上角可开关自动滚动

### UI 重写

- 任务编辑页：分组卡片布局，配置字段两列并排，钩子脚本可折叠，整体更简洁
- 新建环境变量：从弹窗改为底部面板，备注和分组并排显示
- 依赖管理卡片：5 个操作按钮精简为日志图标 + 更多菜单，大幅减少卡片高度
- 系统设置：配置字段两列布局，新增代理设置区块

### 功能同步

- 通知渠道：新增/编辑改为结构化表单（22 种渠道各有专属字段），不再需要手写 JSON
- 订阅管理：新增子目录拉取 (sub_path) 字段，与面板 v2.1.1+ 同步
- Open API：权限范围新增「订阅管理」选项
- 环境变量：启用/禁用 toggle 按钮加大，更易点击

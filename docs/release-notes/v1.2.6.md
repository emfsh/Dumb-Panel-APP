# v1.2.6

## 更新内容

### 优化

- **侧滑方向修正**：环境变量卡片和任务卡片的操作入口改为左滑，卡片向左移动后在右侧露出操作按钮，更符合移动端常见习惯。
- **侧滑提示图标修正**：卡片上的侧滑提示图标同步改为左滑方向，避免用户误以为需要右滑。
- **侧滑返回防误触**：当环境变量或任务卡片的侧滑按钮已经展开时，系统返回手势会先收起侧滑按钮，不会直接退出 APP。

### 修复

- **右滑与系统返回手势冲突**：修复旧版右滑展开操作容易和 Android 返回手势冲突的问题，降低误退出 APP 的概率。
- **回滑关闭误退出**：修复侧滑展开后再回滑收起时，容易误触发系统返回导致 APP 退出的问题。

### 说明

- 本版本只调整 APP 端卡片侧滑交互，不涉及后端接口和数据库结构变更。
- 侧滑操作仍保留原有按钮能力，环境变量包含启用、删除、禁用；任务包含启用/禁用、置顶/取消、复制、编辑、删除。

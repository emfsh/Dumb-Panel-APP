# v1.0.7

## 更新内容

### 新增

- 新增面板日志页面，可在 App 内直接查看面板运行日志
- 新增脚本管理搜索，支持按脚本名称或路径快速筛选
- 新增脚本调试运行入口，无需先创建任务即可直接调试脚本
- 新增任务删除二次确认，可选同时删除关联脚本
- 新增多段 Cron 展示组件，任务列表与详情页可更清晰查看复杂定时规则

### 优化

- 优化脚本保存失败提示，针对路径无效、文件不存在、二进制文件、目录误选、ESM 依赖等场景给出更明确说明
- 优化任务实时日志页面，默认启用自动滚动并在加载历史日志后自动定位到底部
- 优化系统设置与更多页面入口结构，常用运维入口更集中

### 修复

- 修复脚本调试日志查看、停止调试与日志清理相关接口适配问题
- 修复任务实时日志在部分场景下无法及时接入最新输出的问题

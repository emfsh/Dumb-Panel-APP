# 呆呆面板 APP

呆呆面板 Flutter 客户端，用于在移动端管理面板、任务、脚本、环境变量、订阅、依赖、通知、安全设置与备份恢复。

## 当前版本

- APP：`v1.2.2`
- 适配面板：`v2.2.19`

## 本次适配重点

- 支持极验 V4 滑块验证码登录。
- 支持 Python 3.10 / 3.11 / 3.12 多版本依赖环境查看、切换和默认版本设置。
- 支持任务 Python 版本选择，任务默认超时为 `0`。
- 支持 SMTP SSL、备份包下载新版接口、更新镜像源与二进制更新加速源配置。
- 新建任务默认 Python 版本跟随面板依赖管理默认版本。
- 系统更新页适配 Watchtower / 二进制 / Docker 三种更新方式的提示与状态展示。
- 仪表盘在资源采集不可用时不再把内存状态误显示为 `0%`。
- 登录成功后支持 7 天本地可信会话，开启自动登录时可直接进入首页。
- 环境变量、OpenAPI、用户、安全设置、订阅等高频页面补齐了更多真实的成功/失败提示，并统一了多处中文时间显示。
